# ShowHub - Advanced TV Series Management Platform

ShowHub is an enterprise-grade, full-stack application for managing and discovering TV series. It is built using a cloud-native, microservices architecture with advanced design patterns.

## Project Overview

This project implements a comprehensive set of modern technologies and best practices, including:

- **Backend:** Spring Boot 3 microservices with a hexagonal architecture, DDD, CQRS, and Event Sourcing.
- **Frontend:** Micro-frontends using Angular and React.
- **Data Management:** PostgreSQL, MongoDB, Redis, Kafka, and RabbitMQ.
- **Security:** Spring Security 6 with OAuth2/OIDC and Keycloak.
- **DevOps:** Docker, Kubernetes (GKE), Jenkins, Terraform, and Ansible.
- **Observability:** Prometheus, Grafana, ELK Stack, and OpenTelemetry.

This repository contains the source code for all microservices and micro-frontends.
